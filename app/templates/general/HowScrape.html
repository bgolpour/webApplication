What do we mean by Web Scraping?

Web sraping is used to extract data from webpages which they do not provide data in any downloadable formats.

What is the Goal?

Via web scrape applications one could be able to get access to World Wide Web directly using the HTTP, or through a web browser, fetching (or in fact copying) the intended data into a central local database, for later analysis.

Essential Libraries and Tools

  • BeautifulSoup
  • Pymongo
