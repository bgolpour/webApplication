What do we mean by Supervised Learning?

We used supervised learning whenever we want to predict a particular outcome from a given input. In supervised learning we have examples of input/output pairs. With supervise learning we will have a ML model from before said input/output pairs, which comprise our training set.

What is the Goal?

The goal is to make precise predictions for new, never-bofore-seen data.

Which methods are considered as supervised learning?

Below chart is showing a comprehensive pattern of supervised ML methods:

  • k-Nearest Neighbors
    • k-Neighbors classification
    • k-neighbors regression
  • Linear Models
    • Linear regression (aka ordinary least squares)
      • Ridge regression
      • Lasso
    • Linear models for classification
      • Binary Classification:
        • Logistic regression
        • Linear support vector machine
      • Linear models for multiclass classification:
        • one-vs.-rest(A technique to extend a binary classification algorithm to a multiclass classification)
  • Naive Bayes Classifiers
    • GaussianNB
    • BernoulliNB
    • Multinomia
  • Decision Trees
    • Decision Tree Classifier
    • Decision Tree Regressor
    • Ensembles of Decision Trees
      • Random forests
      • Gradient boosted regression trees (gradient boosting machines)
